To run server :

step one:
make sure you have gitbash (or similar) 

step two:
open gitbash to the directory that the server is saved (if you're on windows, you may have to go through the onedrive) (also use `cd` to move around)

step three:
put these in sequence with each other:

npm init
npm install nodemon --save-dev
npm install express 

step four :
add "dev" : "nodemon index.js"

step five :

enter `npm run dev`
to start server
